PHP基础小知识

发表于 2017-05-09 | 分类于 踩坑记录 | | 阅读次数

基础靠慢慢实践和积累的了，在此开篇不连续记载自己PHP开发中的一些基础小知识点。


1、strpos()查找字符串的结果判断
strpos函数是查找子字符串在字符串中的位置，返回值存在为起始位置（int），不存在为false,第一次开发中使用忽视了0和false的判断，必须使用等同操作符（!==)。

<?php
    $a = '韩国,日本，中国，朝鲜是东亚四国';
    $b = '韩国';


    if (strpos($a, $b) !== false){
        echo '$b'是东亚四国;
    }else{
        //如果使用!= 会输出这个结果。
        echo '$b'不是东亚四国;
    }

2、strstr、substr、strrchr、stristr区别
strstr查找匹配A字符串在B字符串中的第一次出现，并返回B字符串剩下的部分,对字符串大小写敏感；substr不匹配，根据给定参数在B字符串中截取字符串；strrchr查找匹配A字符串在B字符串中的最后一次出现，并返回B字符串剩下的部分；stristr不同于stristr的是大小写不敏感。

<?php
    $a = '@';
    $b = '111@222@test.com';

    $result = strstr($b, $a);
    print_r($result); //@222@test.com

    $result1 = strrchr($b, $a);
    print_r($result1); //@test.com

    $result2 = substr($b, 3, (strlen($b) - 3));
    print_r($result2); //@222@test.com

3、php 双引号字符串插入数组值语法问题。

$a = ['a' => 'hello', 'b' => 'world'];
$b = "You say {$a['a']} !"; //正确
$b = "You say $a[a] !";    //正确
$b = "You say $a['a'] !"; //错误
echo $b;

4、永远不要相信前台传入的数据，对前台接收的数据要使用trim,strip_tags,htmlspecialchars进行过滤。

5、合并数组的时候使用array_merge 而不是array_push,后者的结果是嵌套的多位数组。
