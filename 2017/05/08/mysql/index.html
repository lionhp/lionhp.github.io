mysql忘记root密码怎么办

发表于 2017-05-08 | 分类于 踩坑记录 | | 阅读次数

使用phpmyadmin修改本地mysql数据库，结果不小心把密码选择了phpmyadmin加密生成，没有保存密码然后就悲剧了。下面记录下windows系统下mysql忘记root密码的解决办法。

1、停止mysql进程。在命令行下输入net stop mysql(我的环境是phpstudy,命令要使用net stop mysqla),或者简单的打开系统任务管理器在服务里面找到mysql服务右击停止。

2、打开系统命令行工具，切换到mysql的bin目录，然后输入mysqld --skip-grant-tables;
或者
mysqld-nt --skip-grant-tables;
命令执行后切记不要关闭命令行窗口。

3、再打开一个命令行窗口，执行mysql -u root

4、输入代码：

use mysql 
update user set password=password("你的新密码") where user="root"; 
flush privileges; 
exit;

5、然后重启mysql服务，就可以正常登陆mysql数据库了。
