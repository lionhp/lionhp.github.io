foreach,array_walk在PHP5和PHP7中的效率

发表于 2017-05-10 | 分类于 原创 | | 阅读次数

看到技术论坛里有人就php的foreach,array_walk的效率问题有争议，决定自己亲自试验下来证实了。顺便测试下php7比php5性能提升了多少。

1、环境php5.6测试

$num = range(1, 10000);
$start = microtime(true);

foreach ($num as $key => &$value) {
    $value += 1;
}
$end = microtime(true);
$time = $end - $start;
print_r($time); //0.0011529922485352 测试了十来次，基本都在0.001左右。


$num = range(1, 10000);
$start = microtime(true);

array_walk($num, function(&$value){
    $value += 1;
});

$end = microtime(true);
$time = $end - $start;
print_r($time); //0.024404048919678 执行十来次，都在0.02-0.03之间。

2、环境php7.0测试

$num = range(1, 10000);
$start = microtime(true);

foreach ($num as $key => &$value) {
    $value += 1;
}
$end = microtime(true);
$time = $end - $start;
print_r($time); //0.00053811073303223 基本都在0.005左右。


$num = range(1, 10000);
$start = microtime(true);

array_walk($num, function(&$value){
    $value += 1;
});

$end = microtime(true);
$time = $end - $start;
print_r($time); //0.0010521411895752 基本都在0.001-0.014之间。

由上可以看出，foreach在遍历操作数组的效率要比 array_walk 快的，只是两者差距在php7下面要比在php5下面要小，可以看出php7在性能方面提升还是做了很大工作的。不过在数据量不是很大的情况下array_walk代码会比foreach简洁，逼格也更高了。
